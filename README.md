# PulseVote Backend

## Description
This is the Express backend for the PulseVote application.

## Technologies
- Node.js
- Express
- CORS
- Helmet
- dotenv

## Installation

```bash
npm install
```

## Run

```bash
npx nodemon server.js
```

## Endpoints

GET /

Returns:

```
PulseVote API running!
```

GET /test

Returns JSON data used by the frontend.
